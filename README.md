## Description

This is the first team project at Holberton. _printf replicates the C standard library printf() function.

What to learn from thus project:

* Applying varadic functions to a big project
* The complexities of printf
* Managing a lot of files and finding a good workflow
...

## Prototype

int _printf(const char *format, ...);
...

## Usage

* Prints a string to the standard output, according to a given format
* All files were created and compiled on Ubuntu 20.04 LTS using gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
* Returns the number of characters in the output string on success, -1 otherwise
...

## Examples

* _printf("Hello, world\n") prints "Hello world" folloed by a new line
* _printf("%s world", "Hello") prints "Hello world"
* _printf("I am %d years old.", 19) prints "I am 19 years old."
...
